# LyricLab - 高性能卡拉 OK 歌词视频制作与导出工具

`LyricLab` 是一款基于 Web 端的高性能卡拉 OK 歌词视频在线合成与导出工具 (Karaoke Video Maker)。它使用现代 Web 技术，支持用户快速上传音频与歌词文件，定制个性化的歌词视觉动效与背景，并能在浏览器中完成倍速离线压制，一键导出高清 MP4 歌词视频。

---

## 🚀 核心功能亮点

- **音频与字轴歌词解析**：

  - 支持上传主流音频文件，并自动提取和加载波形缓存。
  - 完美支持 `.lrc` 字轴歌词文件（支持精准至字的卡拉 OK 扫光字轴）。
  - 包含未匹配时的自动生成模板与实时在线歌词编辑功能。

- **极致的视觉定制**：

  - **多样化展示模式**：提供单行居中模式、经典的双行交替切换模式、以及三行滚动渐层模式。
  - **高亮扫光效果**：基于 Canvas 线性渐变（`LinearGradient`）与硬切色标（Hard Stop）的高性能扫光机制，边缘渲染干净利落，无抗锯齿重叠白边，已唱区域 100% 遮盖。
  - **灵活的样式调节**：支持字号大小、描边粗细、未唱颜色、已唱高亮色以及字体等全方位的视觉定制。
  - **智能溢出检测**：自动分析每一行歌词在不同字号下的 Canvas 测量宽度，实时提示超出画幅的歌词行，防止导出时歌词被截断。

- **丰富多彩的背景支持**：

  - **动态图片轮播**：支持多图上传，内置平滑的渐变淡入淡出（Fade-in/out）效果以及缓慢缩放、移位的 **Ken Burns 电影感镜头动效**。
  - **视频背景**：支持自定义背景视频上传，采用静音隐式渲染并自动循环，并叠加半透明蒙版以确保歌词清晰度。
  - **纯色背景**：内置纯色背景选项以满足极简主义的设计需求。

- **定制化片头封面**：

  - 可选择启用时长 2.0s 的片头歌曲封面（包含歌曲大标题与歌手副标题），并在 1.5s - 2.0s 自动进行平滑透明度淡出。

- **极速离线视频导出 (WebCodecs)**：
  - 区别于传统的屏幕录制或实时 Canvas 抓流，项目使用 **WebCodecs + MP4 Muxer** 技术。
  - 采用**非实时离线精确逐帧渲染与合轨**方案，压制速度仅受硬件解码性能限制（通常可达实时速度的数倍至数十倍），且保证帧帧精准不丢帧。
  - 支持自定义选择输出分辨率，一键压制成高清 `.mp4` 视频文件。

## 🛠️ 技术栈

- **核心框架**：Vue 3 (Composition API) + TypeScript
- **构建与打包**：Vite + `@vitejs/plugin-vue`
- **CSS 样式引擎**：UnoCSS (高性能原子化 CSS)
- **图标库**：`Iconify` (`@iconify/vue`)
- **视频复用器**：`mp4-muxer`
- **视频编码**：HTML5 WebCodecs API
- **音频控制**：Web Audio API (在线波形分析及音频源接管)

## 📦 本地快速上手

### 1. 克隆并安装依赖

确保您的本地环境已安装 [Node.js](https://nodejs.org/) 与 [pnpm](https://pnpm.io/)。

```bash
# 切换到项目的根目录
cd LyricVideoApp/lyric-video-web

# 安装依赖
pnpm install
```

### 2. 启动本地开发服务

```bash
pnpm dev
```

启动后，在浏览器中访问控制台输出的地址（默认为 `http://localhost:5173`）即可开始创作。

### 3. 项目构建

```bash
# 检查 TypeScript 类型并打包生成静态资源
pnpm build
```
